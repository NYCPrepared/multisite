Adds a button in the rich text editor which generates the shortcode for Contact Form 7 forms.

== Description ==

The plugin adds a button in the richtext editor which allows you to easily generate the short code for *Contact Form 7* WordPress plugin.
It saves you typing the short codes and therefore reducing the chance of typos.
The plugin is very easy to use. You just select a form from a dropdown and it will generate the necessary shortcode.

== Installation ==

You can either search for the plugin from within WordPress or do the manual upload as follows:
1. Upload contact-form-7-ui.zip into to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
